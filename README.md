## 项目描述

本项目是一个基于 Scala + Java + SpringBoot 的 Web 模板，用于与 Scala 相关的 Web 项目快速开发

## 参考场景

- 项目技术栈选择 Scala + SpringBoot
- 原有 Java 项目需要使用 Scala 进行重构，或与 Scala 进行混编
- 微服务体系中使用 Scala 进行数据中台相关的开发

## 项目依赖

- Java：1.8
- Scala：2.11.8
- SpringBoot：1.5.15
